
# Nand2Tetris

[コンピュータの理論と実装](https://www.oreilly.co.jp/books/9784873117126/) における ganyariya の実装ソースコードです。

# How to run

```shell
# ハードウェアシミュレータの場合
./nand2tetrisSoftware/tools/HardwareSimulator.sh
```

#### Online での実行

Online で実行できる環境が用意されている
そのためこちらを利用しても問題ない

https://nand2tetris.github.io/web-ide/cpu
